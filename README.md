## ETL‑pipeline: обработка данных онлайн‑университета → PostgreSQL → Google Sheets → email‑оповещение ##
### Цель проекта
Создать автоматизированный процесс, который:

- получает данные из API онлайн‑университета,
- логирует каждую запись,
- очищает и нормализует данные,
- загружает их в PostgreSQL,
- агрегирует данные в Google Sheets,
- отправляет email‑уведомление о статусе загрузки.


Проект имитирует реальный рабочий процесс EdTech‑платформы и демонстрирует навыки API‑интеграции, ETL, логирования, работы с Google API и SMTP.

### Описание логики проекта
1. Получение данных из API + логирование (api_data_and_log.py)

Скрипт:

- отправляет запросы к API LMS,
- получает данные о попытках пользователей,
- приводит их к нужному формату,
- записывает каждую попытку в лог‑файл (logs/),
- загружает данные в PostgreSQL.

Формат данных:

|Поле |Описание |
| ----- | ----- |
|user_id |ID пользователя |
|oauth_consumer_key |токен клиента |
|lis_result_sourcedid |ссылка на блок в LMS |
|lis_outcome_service_url |URL для отправки оценки |
|is_correct |успешность попытки |
|attempt_type |run или submit |
|created_at |дата и время|

2. Агрегация данных и выгрузка в Google Sheets (google_sheets.py)

Скрипт:

- читает данные из БД,
- агрегирует их (например: количество попыток, процент успешных, активность по датам),
- загружает итоговую таблицу в Google Sheets через Google API.


Используется для мониторинга активности студентов и качества контента.

3. Email‑оповещение (sendemail.py)

Скрипт:

- формирует отчёт о загрузке данных,
- подключается к SMTP,
- отправляет письмо (например: «Данные успешно обновлены», «Ошибка при загрузке»).


Это имитирует реальный мониторинг ETL‑процессов.

### <b><i> Запуск проекта </i></b> ###
1. Клонировать репозиторий
```
git clone https://github.com/annagorelove/project_python.git
```

2. Установить зависимости
```
pip install -r requirements.txt
```

3. Настроить config.ini

Указать:

- доступы к БД
- API‑ключи
- путь к логам
- параметры Google API
- SMTP‑настройки


4. Получить данные из API
```
python api_data_and_log.py
```

5. Агрегировать данные и выгрузить в Google Sheets
```
python google_sheets.py
```

6. Отправить email‑уведомление
```
python sendemail.py
```
