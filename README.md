## Обработка данных онлайн-университета c выгрузкой в базу данных, агригация данных в Google Sheets и настройка оповещения на почту ##
### <b><i> Описание работы скрипта </i></b> ###
Скрипт обращается к апи и обрабатывает полученные данные для дальнейшей выгрузки в бд, логирует каждую такую запись в папку <b><i> logs </i></b>. Затем агрегируют эти данные для выгрузки в Google Sheets и отправляет оповещение на почту.

Задача в рамках проекта - настроить обработку данных через апи и отформатированные данные выгрузить в бд (а также написать скрипт для агригации данных и отправки оповещения на почту).

Формат выгрузки:
- <b> user_id </b> - строковый айди пользователя;
- <b> oauth_consumer_key </b> - уникальный токен клиента;
- <b> lis_result_sourcedid </b> - ссылка на блок, в котором находится задача в ЛМС;
- <b> lis_outcome_service_url </b> - URL адрес в ЛМС, куда мы шлем оценку;
- <b> is_correct </b> - была ли попытка верной (null, если это run);
- <b> attempt_type </b> - ран или сабмит;
- <b> created_at </b> - дата и время попытки.

### <b><i> Структура проекта </i></b> ###
Проект содержит файлы имеющие:
- <b> api_data_and_log.py </b> - содержит скрипт логирования записей, функцию для обращения к апи и обработки этих данных в нужном формате, а так же подключение к бд и выгрузку этих данных в PostgresSQL.  <n>  
По результату работы скрипта генерируются логи, которые потом добавляются в папку logs.
- <b> google_sheets.py </b> - скрипт для агригации полученных данных через апи и загрузки данных в созданную таблицу Google Sheets;
- <b> sendemail.py </b> - скрипт для подключения к email и отправка отчета об выгрузке данных;
- <b> requirements.txt </b> - файл с установленными библиотеками, необходимыми для работы данного проекта.

### <b><i> Запуск проекта </i></b> ###
Для запуска проекта необходимо загрузить репозиторий на свой компьютер:

     [git clone https://github.com/annagorelove/project_python.git]

Далее установить используемые в проекте библиотеки из файла <b><i> requirements.txt </i></b>:

     pip install -r requirements.txt

Далее необходимо запустить файл <b><i> api_data_and_log.py </i></b> для получения данных из апи.

По итогу добавления логов в папку <b><i> logs/ </i></b> запустить скрипт <b><i> google_sheets.py </i></b>, который агрегирует все данные и выгрузит их в Google Sheets. И затем запустить скрипт <b><i> sendemail.py </i></b> для отправки оповещения на почту.
